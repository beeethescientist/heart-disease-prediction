# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview

Heart disease is one of the leading causes of death worldwide. Early prediction can assist healthcare professionals in identifying high-risk patients and supporting timely medical intervention.

This project applies machine learning techniques to predict the likelihood of heart disease using patient health indicators. The project demonstrates the complete machine learning workflow, from data preprocessing to model evaluation.

---

## 🎯 Objectives

- Explore and understand the dataset
- Clean and preprocess the data
- Train multiple machine learning models
- Evaluate model performance
- Select the best-performing model for heart disease prediction

---

## 📊 Dataset

The dataset contains patient medical information commonly used to assess heart disease risk, including:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG Results
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression (Oldpeak)
- Slope of Peak Exercise ST Segment
- Number of Major Vessels
- Thalassemia
- Target (Heart Disease)

---

---

## ⚙️ Project Workflow

### 1. Data Exploration

- Loaded the dataset
- Examined data structure
- Checked for missing values
- Explored feature distributions

### 2. Data Preprocessing

- Cleaned the dataset
- Prepared features for machine learning
- Split the data into training and testing sets

### 3. Model Training

Several machine learning algorithms were trained and compared to identify the most suitable model.

### 4. Model Evaluation

Models were evaluated using standard classification metrics to determine prediction performance.

### 5. Model Deployment

The best-performing model was saved using Joblib for future use in a prediction application.

---

## 📈 Results

The project successfully identified a machine learning model capable of predicting heart disease based on patient characteristics.

The trained model was exported for deployment in a Python application.
